Linda: Social Logic Interface for ASD
Linda is a specialized web application designed to support autistic students—specifically those navigating the high-stimulus environment of university life. Linda acts as a "Logical Translator," converting the unpredictable "noise" of social cues into predictable, rule-based data to help users recognize emotions and build social competence.

Core Purpose
For individuals like Mona, social interactions are often overwhelming due to alexithymia and sensory overload. Linda removes the "vague" and "implied" layers of communication by offering:

Structured Scenario Quizzes: Learning emotion recognition through clear, visual patterns.

Predictable Logic: A rigid, rule-based interface that prevents decision paralysis.

Energy-Aware Design: Built-in safeguards to monitor and prevent social exhaustion (burnout).

Key Features
Scenario-Based Learning: Interactive quizzes using 7 universal emotion models.

Failure Protocol (Module 4): Sophisticated, patient handling of confusion, silence, and distress.

Safe-Mode Logic: Immediate escalation for unsafe requests or medical/emergency concerns.

Non-Negotiable Boundaries: Strict adherence to literal language, 30-second processing buffers, and binary-choice decision-making.

Technical Design Principles
Predictability: The interface never uses sarcasm, idioms, or vague instructions.

No Rushing: Every interaction respects a minimum 60-second processing buffer.

Low Cognitive Load: Strictly limited to 3 choices per screen to avoid burnout.
